# 🏥 30 Days SQL Micro Course — Hospital Data Analysis Assignment

## 📌 About This Project

This repository contains my assignment solution for the **30 Days SQL Micro Course** designed by **Satish Dhawale**, Founder of [Skill Course](https://www.skillcourse.in).

In this assignment, I used SQL queries to extract key insights from a hospital dataset (`hospital.csv`), covering real-world scenarios like patient statistics, medical expense analysis, and department-wise performance.

## 🗄️ Dataset

**File:** `hospital.csv`

The dataset contains information about hospitals including:
- Hospital name and location
- Department details
- Patient count and doctor count
- Admission and discharge dates
- Medical expenses

---

## 📝 SQL Queries Covered

| # | Topic |
|---|-------|
| 1 | Total number of patients across all hospitals |
| 2 | Average number of doctors per hospital |
| 3 | Top 3 departments with the highest number of patients |
| 4 | Hospital with the maximum medical expenses |
| 5 | Daily average medical expenses per hospital |
| 6 | Longest hospital stay (using date difference) |
| 7 | Total patients treated per city |
| 8 | Average length of stay per department |
| 9 | Department with the lowest number of patients |
| 10 | Monthly medical expenses report |

---

## 🛠️ SQL Concepts Used

- `SUM()`, `AVG()`, `MIN()`, `MAX()` — Aggregate functions
- `GROUP BY` — Grouping data
- `ORDER BY` — Sorting results
- `LIMIT` — Restricting output rows
- `EXTRACT()` — Extracting month/year from dates
- Date arithmetic — Calculating stay duration
